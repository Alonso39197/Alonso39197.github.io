Introduction

Que es el html?

El HTML es un lenguaje de marcado, no para estructurar una pagina web.

Explica que son los tags de head y body.

El tag head, sirve para cosas que no son visibles para el usuario, como lo es el titulo, los links a los css y scripts

En cambio, el body contiene todo lo que es visible para el usuario, textos, imagenes, links, etc.

Basic tags

Un section tag representa una sección cualquiera en una pagina web, estas deben de tener un titulo para definirlas con solo unas excepciones. Un article tambien es parecido a un tag de sección porque son elementos semanticos, pero este puede existir por si solo, esto quiere decir que el contenido dentro tiene una estructura que lo hace funcionar incluso si lo sacas de la pagina web y lo agregas a otra. El aside representa una seccion de una pagina que el contenido que estan relacionadas con el contenido principal, aunque sea indirectamente. El header es el contenido introductorio de una pagina web, normalmente contiene el titulo de la pagina, el logo, el menu de navegacion, etc. El footer es el contenido que se encuentra al final de una pagina web, normalmente contiene informacion de contacto, derechos de autor, etc.

CSS

What is CSS? Es un lenguaje de estilo para definir la parte visual o la apariencia de una pagina web.

What is the purpose of CSS? El proposito de CSS es separar el contenido de una pagina web de su diseño, esto para que el contenido sea mas facil de acceder y tiene un control del diseño

How do you apply a CSS to an HTML (explain the 3 options) 1. Con un archivo CSS externo 2. Con un CSS interno, es decir la etiqueta style dentro del head 3. Con un CSS en linea, es decir con el atributo style dentro de la etiqueta del elemento que se utilizara

What is a selector? What is a CSS rule? Un selector es el elemento al que se le aplicara el estilo, puede ser una etiqueta, una clase o un id

Basic CSS

Que es un tag selector? Es un selector que selecciona todos los elementos de un tipo especifico, por ejemplo, el selector p selecciona todos los elementos p de la pagina web

Que es un class selector? Es un selector que selecciona todos los elementos que tienen una clase especifica, por ejemplo, el selector de .titulo selecciona todos los elementos que tienen la clase titulo

Que es un id selector? Es un selector que selecciona un elemento con un id especifico, por ejemplo, el selector de #titulo selecciona el elemento que tiene el id titulo

What is a class? What is an id? How are they related to CSS? Una clase es un atributo que se puede usar en varios elementos HTML para agruparlos y aplicarles los mismos estilos. En cambio, un id es único y se usa para identificar un elemento específico dentro de la página. Ambos se utilizan en CSS para seleccionar y aplicar estilos de manera más precisa.

Explain the CSS Box Model? El modelo de caja de CSS describe cómo cada elemento HTML se comporta como una caja compuesta por contenido, padding, borde y margen. El contenido es lo que se muestra, el padding es el espacio interno, el borde lo rodea y el margen lo separa de otros elementos. Este modelo ayuda a controlar el tamaño y la distribución en la página.

Explain the media queries Las media queries son una característica de CSS que permite aplicar estilos específicos según las características del dispositivo o la ventana del navegador, como el ancho, la altura, la orientación, etc. Esto es fundamental para el diseño web responsivo, ya que permite adaptar el diseño a diferentes tamaños de pantalla y dispositivos.
